* 元韵码 yym

(使用简化元音，故称“元韵码”)

1.一码声母

2.二码：

韵母定列：参考[[https://github.com/macroxue/shuangpin/?tab=readme-ov-file#%E7%81%AB%E6%98%9F%E5%8F%8C%E6%8B%BC][火星双拼]]布局，经过省略韵头和同元音合并,ü并为u，所有韵母可以分为10列韵母。

声调定排：第一行1和轻声，第二行2和3声，第三行4声，Q和P还包括4声
